### 1.2 nullable과 non-nullable

```dart

📌 nullable : null 허용
📌 non-nullable : null 불허용
📌 null : 값이 없음
```

### 1.3 final과 const

```dart
/*
📌 final : 한번 할당되면 변경 불가
📌 const : 한번 할당되면 변경 불가, 빌드 시점에 값을 알고 있어야 함
@see https://dart-ko.dev/language/variables#final-const
*/
void main() {
  // final String name = '메시';
  final name = '메시'; // 타입 생략 가능
  print(name);

  // const String name2 = '호날두';
  const name2 = '호날두'; // 타입 생략 가능
  print(name2);
}
```

## 2. 연산자

- cf. https://dart-ko.dev/language/operators

### 2.1 정수 산술 연산자

```dart
void main() {
  int number = 2;

  print(number + 2);
  print(number - 2);
  print(number * 2);
  print(number / 2);

  print('--------------------');
  print(number % 2); // 0
  print(number % 3); // 2
  print(number); // 2

  print('--------------------');
  number ++;
  print(number); // 3

  number --;
  print(number); // 2
}
```

---

### 2.2 실수 산술 연산자

```dart
void main() {
  double number = 4.0;
  print(number); // 4

  number += 1;
  print(number); // 5

  number -= 1;
  print(number); // 4

  number *= 2;
  print(number); // 8

  number /= 2;
  print(number); // 4
}
```

---

### 2.3 널 조건 연산자

```dart
void main() {
  double? number = 4.0;
  print(number); // 4

  number = 2.0;
  print(number); // 2

  number = null;
  print(number); // null

  // number가 null이면, 3.0으로 초기화
  number ??= 3.0;
  print(number); // 3
}
```

```dart
}
```

## 3. 자료구조

### 3.1 리스트(List)

```dart
void main() {
  // 📌 리스트(List)
  List<String> team = ['Messi', 'Ronaldo', 'Neymar', 'Mbappe'];
  List<int> numbers = [1, 2, 3, 4, 5];

  // 📌 순서(index), 0부터 시작
  print(team[0]); // Messi
  print(numbers[2]); // 3

  // 📌 길이(length)
  print(team.length); // 4
  print(numbers.length); // 5

  // 📌 추가(add)
  team.add('Suarez');
  print(team); // [Messi, Ronaldo, Neymar, Mbappe, Suarez]

  // 📌 삭제(remove)
  team.remove('Ronaldo');
  print(team); // [Messi, Neymar, Mbappe, Suarez]

  // 📌 값의 인덱스 위치(indexOf)
  print(team.indexOf('Mbappe')); // 2
}
```

---

### 3.2 맵(Map)

```dart
void main() {
  // 📌 Map (Key, Value 형태)
  Map<String, String> team = {
    '바르셀로나': '메시',
    '레알마드리드': '호날두',
    '맨체스터시티': '홀란드',
  };
  print(team);

  Map<String, bool> isTeam = {
    '바르샤': true,
    '레알': true,
    '맨시티': false,
  };
  print(isTeam);

  // 📌 Map 추가
  isTeam.addAll({
    '맨유': false,
  });
  print(isTeam);
  print(isTeam['맨시티']); //

  isTeam['아스날'] = false;
  print(isTeam);

  isTeam['맨유'] = true;
  print(isTeam);

  print('---------------------------');
  // 📌 Map 삭제
  isTeam.remove('맨시티');
  print(isTeam);

  print('---------------------------');
  // 📌 key값만, values값만 가져오기
  print(isTeam.keys);
  print(isTeam.values);
}
```

### 3.3 Set

- `Map` : Key, Value 형태의 자료구조, 중복 허용
- `Set` : 중복을 허용하지 않는 자료구조

```dart
void main() {
  // 📌 Set
  final Set<String> names = {'메시', '호날두', '네이마르'};
  print(names);

  // 📌 add : 추가
  names.add('베일');
  print(names);

  // 📌 remove : 삭제
  names.remove('메시');
  print(names);

  // 📌 contains : 존재여부를 boolean값으로 반환
  print(names.contains('호날두'));
}
```

## 7. 함수

### 7.1 함수 선언

```dart
void main() {
  addNumbers(10, 20, 30);
  addNumbers(10);
}

// 3개의 숫자를 더하고 짝수인지 홀수인지 판별하는 함수
// 📌 positional parameter : 순서가 중요한 파라미터
// 📌 optional parameter : 있어도 되고, 없어도 되는 파라미터
addNumbers(int x, [int y = 20, int z = 50]) {
  int sum = x + y + z;

  print('x: $x');
  print('y: $y');
  print('z: $z');

  if (sum % 2 == 0) {
    print('짝수');
  } else {
    print('홀수');
  }
}
```

#### 7.1.1 named parameter

```dart
void main() {
  addNumbers(y: 10, x: 20);
  addNumbers(x: 10, y: 20, z: 30);
}

// 3개의 숫자를 더하고 짝수인지 홀수인지 판별하는 함수
// 📌 named parameter : 이름이 있는 파라미터(순서가 중요하지 않음)
addNumbers({
  required int x,
  required int y,
  int z = 30,
}) {
  int sum = x + y + z;

  print('x: $x');
  print('y: $y');
  print('z: $z');

  if (sum % 2 == 0) {
    print('짝수');
  } else {
    print('홀수');
  }
}
```

#### 7.2 void, return

```dart
// void : 공허한, 반환값이 없는
void main() {
  int result = addNumbers(y: 10, x: 20);
  int result2 = addNumbers(x: 10, y: 30, z: 40);

  print('result: $result');
  print('result2: $result2');

  print('sum: ${result + result2}');
}

int addNumbers({
  required int x,
  required int y,
  int z = 30,
}) {
  int sum = x + y + z;

  print('x: $x');
  print('y: $y');
  print('z: $z');

  if (sum % 2 == 0) {
    print('짝수');
  } else {
    print('홀수');
  }

  return sum;
}
```

## 8. 타입 선언(TypeDef)

```dart
void main() {
  Opertaion operation = add;
  int result = operation(10, 20, 30);
  print(result); // 60

  operation = subtract;
  int result2 = operation(10, 20, 30);
  print(result2); // -40

  int result3 = calculate(30, 40, 50, add);
  print(result3); // 120

  int result4 = calculate(40, 50, 60, subtract);
  print(result4); // -70
}

// signature
typedef Opertaion = int Function(int x, int y, int z);

// 더하기
int add(int x, int y, int z) => x + y + z;

// 빼기
int subtract(int x, int y, int z) => x - y - z;

// 계산
int calculate(int x, int y, int z, Opertaion operation) {
  return operation(x, y, z);
}
```
