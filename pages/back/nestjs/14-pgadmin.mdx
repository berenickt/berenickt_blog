## 2. PgAdmin 사용법

pgAdmin 4를 킨 뒤에, Add New Server를 클릭한다.

- Gerneral
  - Name : NestJS Postgres Server
- Connection
  - Host name : 127.0.0.1
  - Port : 5808
  - Maintenance : postgres
  - Username : postgres
  - Password : postgres
- 저장

왼쪽 맨 위에 postgres를 누른 상태에서 Query Tool 선택하면 쿼리 입력 가능하다.
