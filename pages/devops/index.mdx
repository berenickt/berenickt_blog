# 데브옵스(DevOps) 로드맵

## 1. 1~4주차

### 1.1

- 클라우드가 서비스가 생겨난 이유
- 클라우드 등장 이전은 어땠는지
- 어떤 불편함을 해소하고자 클라우드가 등장했을까?
- 도커, 쿠버네티스, 테라폼 금지❌
  - 눈 돌아감
  - 왜 쓰지는 모른 채로 배우게 됨

### 1.2

- AWS 사용해보기
- AWS 계정 생성 / 신용카드 등록
- AWS EC2 사용해보기
