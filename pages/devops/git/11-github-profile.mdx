## 2. Github 내 이동 명령어

- Find Mode : `/`
  - 검색창이 눌려짐
- File Finder : `T`
  - 파일 탐색 모드가 시작됨
  - 방향키로 탐색 가능
- Command Paletee :
  - Window : `Ctrl + K`
  - Mac : `Command  + K`
  - 더 세부적인 repository를 알고싶다면, repository로 키보드로 이동해서 `Tab 키`
  - `>`누르면 새로운 repository 생성 등 가능
- Jump to a Line : `L`
  - 이동할 라인 선택
- Keyboard Shortcus : `?`
  - 어떤 단축키가 있는지 확인
