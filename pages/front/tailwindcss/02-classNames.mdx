## 1. classNames

### 1.1 설치

```bash
yarn add classnames
```

---

### 1.2 간단한 사용법

```tsx
import classNames from 'classnames'

classNames('one', 'two') // = ‘one two‘
classNames('one', { two: true }) // = ‘one two‘
classNames('one', { two: false }) // = ‘one‘
classNames('one', ['two', 'three']) // = ‘one two three‘

const myClass = 'hello'
classNames('one', myClass, { myCondition: true }) // = ‘one hello myCondition'
```

이런 식으로 여러 가지 종류의 파라미터를 조합해 CSS 클래스를 설정할 수 있기 때문에,
컴포넌트에서 조건부로 클래스를 설정할 때 매우 유용하다.

```tsx
/**
 * highlighted 값이 true면, highlighted 클래스가 적용
 * theme으로 전달받는 문자열은 내용 그대로 클래스에 적용
 */
const MyComponent = ({ highlighted, theme }) => (
  <div className={classNames('MyComponent', { highlighted }, theme)}>Hello</div>
)
```

props 값에 따라 다른 스타일을 주기가 쉽다.

반대로 위 코드를 classnames를 쓰지 않고 적용해보면 다음과 같다.

```tsx
const MyComponent = ({ highlighted, theme }) => (
  <div className={`MyComponent ${theme} ${highlighted ? 'highlighted' : ''}`}>Hello</div>
)
```
