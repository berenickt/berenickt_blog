## 3. 주요 기능

- setting → sticky 설정해두기
- Settings Sync (설정한 것들 동기화하기)

### Command Palette

- (Windows/Linux) : `Ctrl + Shift + P` 또는 F1
- (Mac) : `Shift + Command + P`

---

### Quick Open

- (Windows/Linux) : `Ctrl + P`
- (Mac) : `Command + P`

---

### User Settings

- (Windows/Linux) : `Ctrl + ,`
- (Mac) : `Command + ,`

---

### Toggle Sidebar

- (Windows/Linux) : `Ctrl + B`
- (Mac) : `Command + B`

---

### Toggle Terminal

- (Windows/Linux) : `Ctrl` + (틸드)
- (Mac) : `control` + (틸드)

---

### 빠른 코드 이동 모음

- beginning /end of file
  - (Windows/Linux) : `Ctrl + Home / Ctrl + End`
  - (Mac) : `Command + ↑ / Command + ↓ `
- beginning / end of line : 라인 맨처음 끝으로 이동
  - (Windows/Linux) : `Home / End`
  - (Mac) : `Command + ← / Command + → `
- Move Word : 단어별 이동
  - (Windows/Linux) : `Ctrl + ← / Ctrl+ →`
  - (Mac) : `Command + ← / Command + → `
- Select Word : 단어 한번에 선택
  - (Windows/Linux) : `Ctrl + Shfit + ← / Ctrl + Shfit + →`
  - (Mac) : `Command + ← / Command + → `
- Copy Line
  - (Windows/Linux) : `Ctrl + c`
  - (Mac) : `Command + c `
- Move line down/up
  - (Windows/Linux) : `Alt + ↑ / ↓`
  - (Mac) : `Command + ↑ / ↓ `
- Insert line below : 작성 중인 줄에서 바로 다음 줄로 넘어갈 떄
  - (Windows/Linux) : `Ctrl + Enter`
  - (Mac) : `Command + Enter `

---

### 단어 다중선택

- Multi Selection : 한 파일 안에서 여러 번 반복되어 사용되는 단어를 1개씩 선택
  - (Windows/Linux) : `Ctrl + D` (해당되는 단어 위에서)
  - (Mac) : `Command + D `

---

### 이전커서로 이동

- Undo last Cursor
  - (Windows/Linux) : `Ctrl + U`
  - (Mac) : `Command + U `

---

### 다중 커서 선택 모음

- Insert Cursor
  - (Windows/Linux) : `Alt + Click`
  - (Mac) : `Option + Click `
- Column (box) selection
  - (Windows/Linux) : `Shift + Alt + drag mouse`
  - (Mac) : `Command + option + drag mouse `
- 마우스로 여러 줄 선택
  - (Windows/Linux) : `Ctrl + Alt + ↑ / ↓ `
