## 1. 다양한 연결 동사

```
Steve is happy. (그냥 스티브는 행복하다)
```

be 동사 외의 연결 동사들은 자신이 가진 의미를 덧붙이며 다음 정보로 연결합니다.

```
Steve looks happy. (스티브는 행복해 보인다)
Steve feels happy. (스티브는 행복감을 느낀다)
Steve gets happy.  (스티브는 안행복했다가 행복해졌다)
```

be 동사 이외 연결 동사들은 동작 동사로 쓰이기도 합니다.
굳이 둘은 구분하는 방법은 연결 동사로 쓰인 경우는 be동사로 교체해도 의미가 통합니다.
반면에 동작 동사로 쓰인 경우는 be동사로 교체해도 의미가 통하지 않습니다.

```
Steve feels happy.    (실제 동작이 없음, 연결 동사, be동사로 교체 가능)
Steve feels the wall. (실제 동작이 있음, 동작 동사)
```

그렇지만 별로 실용적으로 쓰이지는 않습니다.
그냥 '연결 동사'로도 쓰일 수 있다는 것을 생각하며 맥락에 맞게 의미를 자연스럽게 파악하면 됩니다.

대표적인 연결 동사들을 보면 다음과 같습니다.

| 1. 감각 동사들 | 2. 뭔가 변하는 동사들 | 3. 뭔가 유지하는 동사들 |
| :------------: | :-------------------: | :---------------------: |
|      look      |          get          |          stay           |
|      seem      |        become         |         remain          |
|     appear     |         grow          |                         |
|      feel      |         turn          |                         |
|     sound      |                       |                         |
|     smell      |                       |                         |
|     taste      |                       |                         |

## 2. 동작 동사 get과 연결 동사 get 예시

```
// 동작 동사 get
I got a letter from Steve.
Steve got a cold
Steve didn't get the joke.

// 연결 동사 get
As you get old, your memory gets worse.
My watch got broken yesterday.
Steve got to Seoul yesterday.
```

암기할 필요없이 (존재나 상태가 변하는) + (뒤에 나오는 단어)를 조합하면 됩니다.

- get up
- get well
- get ready
- get to do
- get moving
- get married

일반적으로 학교에서 동사를 배울 때는 자동사, 타동사, 수여동사, 지각동사, 사역동사 등 많이 구분하는데,
실제 영어 동사는 2가지로 구분할 수 있습니다. 즉, 행위자로 시작하는 것은 경우에 따라 2가지로 나눕니다.

1. 동작 동사 (Action Verb)
2. 연결 동사 (Linking Verb)

```
(동작 동사) 행위자 → 동작 → 대상 → 상황 정보
(연결 동사) 행위자 → 연결 동사 → 존재·상태 정보
```

> 원어민들이 쓰는 동사를 빅데이터를 이용해 분석해보면 동작 동사 약 50%, 연결 동사 약 50%이라고 합니다.
