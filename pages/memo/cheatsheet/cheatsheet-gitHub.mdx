## 1. GitHub 단축키

- Repository 안의 파일 검색 : `단축키 T`
- 전체 Repository에서 찾기 : `단축키 /`
- 소스 수정한 사람 찾기 : `단축키 B`
  - Git Blame 명령어
- Command Palette :
  - Mac : `Cmd + k`
  - Windows : `Ctrl + K`
- Hide Comments : `단축키 i`
- Jump to a Line : `단축키 L`
- Keyboard Shortcuts : `?`
- 퍼머링크 : `Github 라인넘버 위에 클릭 후 Shift 누른 상태에서 다른 라인넘버를 클릭`
  - 퍼머링크를 영구적인 URL로 변경
  - Canonical URL : `단축키 Y`
- cf. [(43) 깃허브 사용법 + 단축키 (이것좀 알고 사용하자 🔥) - YouTube](https://www.youtube.com/watch?v=2tK0txsNd6U)
